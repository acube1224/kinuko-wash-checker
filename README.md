# 長襦袢セルフ判定アプリ

## 概要
長襦袢を自宅で洗ってよいかどうかを、7つの質問に答えることで3段階で判定するセルフチェックWebアプリです。  
「日々、絹子さん。」ブランドの長襦袢専用洗剤「洗う絹子さん」の販促・顧客支援ツールとして設計されています。

---

## 完成済みの機能

### 画面構成（8画面）
1. **トップ画面** - サービス紹介・診断開始
2. **案内画面** - 診断前の説明・「わからない」で進める案内
3. **質問画面** - 7問を1画面1問で表示・進捗バー付き
4. **ローディング画面** - 判定演出
5. **判定結果画面** - 3段階バッジ・理由チップ・行動ボタン
6. **判定理由の詳細画面** - 回答ごとの影響をリスト表示
7. **洗う前の注意点画面** - 6項目の注意チェックリスト
8. **次の行動画面** - 相談・商品導線・再診断

### 7問の質問
1. 素材（正絹/絹混/化繊・木綿・麻/不明）
2. 生地の特徴（なめらか/ちりめん/絽など極薄/不明）
3. 仕立て（ひとえ単/ふたえ/異素材あわせ/不明）
4. 繊細な加工の有無（金彩・箔・ラメ・刺繍）
5. 水処理歴（水通し・丸洗い・洗い張りの有無）
6. 過去の洗いの変化（縮み・ヨレ・色落ち・風合い変化）
7. 地色（白・淡色/中間色/濃色/多色・ぼかし）

### 判定ロジック（ルールベース）
- **A判定**（洗える候補）: スコア≦2
- **B判定**（慎重に検討）: スコア3〜7
- **C判定**（自宅洗い非推奨）: スコア≧8

#### 強制ルール
- ちりめん・強シボ → 即C判定
- 化繊・木綿・麻 → 即A判定
- 過去に縮み/色落ち実績あり → 最低B
- 繊細加工あり → 最低B

## WordPressへの埋め込み方法

### 埋め込みコード（固定ページのHTMLブロックに貼る）

```html
<!-- 長襦袢セルフ判定 埋め込みコード -->
<iframe
  id="kinuko-iframe"
  src="ここに公開URLを入れる"
  width="100%"
  height="800"
  style="border:none; width:100%; display:block;"
  scrolling="no"
  title="長襦袢セルフ判定"
></iframe>

<script>
// iframeの高さを自動調整するスクリプト
window.addEventListener('message', function(e) {
  if (e.data && e.data.type === 'kinuko-resize') {
    var iframe = document.getElementById('kinuko-iframe');
    if (iframe) {
      iframe.style.height = (e.data.height + 20) + 'px';
    }
  }
  // 画面遷移時にページ上部へスクロール
  if (e.data && e.data.type === 'kinuko-scroll-top') {
    var iframe = document.getElementById('kinuko-iframe');
    if (iframe) {
      iframe.scrollIntoView({ behavior: 'smooth', block: 'start' });
    }
  }
});
</script>
```

### 手順
1. GenSparkでアプリを**Publish（公開）**してURLを取得
2. WordPressの固定ページを編集
3. **「カスタムHTML」ブロック**を追加
4. 上記コードの `src="ここに公開URLを入れる"` 部分に公開URLを入力
5. 更新して確認

### レスポンシブ対応
| 端末 | 表示幅 | 動作 |
|---|---|---|
| スマホ | ~480px | 全幅・シンプルレイアウト |
| タブレット | 481~768px | カード風・余白あり |
| PC | 769px~ | 最大幅600px・中央寄せ |

### 注意点
- WordPressテーマによっては `iframe` のスタイルが干渉する場合があります
- その場合は `style="border:none; width:100% !important;"` と `!important` を追加してください
- Cocoon・Swell・Lightning などの主要テーマで動作確認推奨です

## 今後の推奨開発ステップ

1. **画像アップロード機能の追加** - スマホカメラで撮影した生地写真をAIで補助判定
2. **相談フォームの接続** - 結果Cや不安の強いユーザーを専門家相談へ誘導
3. **商品導線の強化** - 結果Aに「洗う絹子さん」購入ページへのリンク追加
4. **判定ログの収集** - ユーザー回答データを蓄積して精度改善に活用
5. **SNSシェア機能** - 「洗える候補でした！」等のシェア導線
6. **正規取扱店一覧との連携** - C判定ユーザーを近隣の相談可能な呉服店へ誘導
